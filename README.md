# 7 Wonders

## Set-up for working on this project.

The server is written in Node.js. You will need to install the Node.js runtime from the [Node.js homepage](https://nodejs.org). A Visual Studio 2017 project is included which includes the npm dependencies.

When the server is running in the default environment, the client is available at [localhost:1337](http://localhost:1337). The source for the client is in the public folder.

When the server is running in the default environment, a simple client for testing the server is available at [localhost:1337/test](http://localhost:1337/test). The source for the test client is in the test folder. It is developed with AngularJS and __not__ any recent version of Angular since it's simply intended to be a very light wrapper around the API calls.
